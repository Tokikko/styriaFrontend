## Prerequisites


https://nodejs.org/en/

## Instalation

Clone project:

```
git clone https://github.com/Tokikko/styriaFrontend.git
```

Install packages:

```
npm install
```

Run the app:

```
npm start
```

Go to
```
http://localhost:3000/
```

To view the site 

*Just make sure that the server(styriaBackend) is running or the app will look quite empty
